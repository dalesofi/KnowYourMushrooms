# Process Followed 

1. Explore possible datasets in Kaggle according to personal curiosity 
2. Choose a dataset according to requirements 
3. Create Git repository, README, Exploratory notebooks 
4. Install all dependencies and libraries at the top
5. Load the dataset into Pandas and explore (shape, info, describe, head, nans, see all unique values,)
6. Data Cleaning: Drop some columns full of nans (4), check for duplicates
7. Basic EDA: check for class imabalance in target variable, basic stats, Plots: histograms, scatterplots, barplots, correlation matrix, heatmap, boxplots. 
8. Preprocessing: Target variable encoding, Label encoding, scaler, fill nans w KNN 
9. Train Test Split 
10. Choosing the model 
11. Hyperparameter tunning 
12. Presentation 
13. Feedback and iteration
